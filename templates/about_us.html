This is our Cancer Incidences dataset navigation software: the W. A. T. C. H. - Web Analysis Tool (of) Cancer History.

The purpose of this system is to guide you in the journey of further discovering the intricacies of out group's dataset. Specifically, this is about the number of occurences of cancer within the U.S.

The data is categorized by:

  • The [State] it happened in
  • The [Year] it happened (we only have the cases from the year 2000 to 2021)
  • The leading [Site] on the body where it happened
  • The [Sex] of the person unfortunately inflicted with cancer
  • Plus, a [Count] of occurences of each unique combination of the above factors.

Currently, our team has created a demo to show you how a search for the number of [Liver] cancer cases occuring in [Texas], [2023] suffered by now [Dead] people using the 'and' combination. Only cases that match all provided valid data will be counted. The full version will also have an 'or' combination option, where cases that has any matching data at all will be counted even if only 1 category of theirs matches.

This combination will search through our dataset for any cancer cases that matches with all of the data located in the square brackets. Invalid inputs, such as [Dead], will be ignored by the system and reported to you.
